Игра "Лабиринт" на c++

Консольная игра с генерацией случайных лабиринтов.

Разработчик: Сюткин Дмитрий
Курс/группа: ИТ-13,14-2025
Преподаватель: Кухар Д.А.
Дата реализации: 15.01.2026

Функциональность:
• генерацию лабиринта
• поиск выхода через волновой алгоритм
• восстановление пути
• сохранение и загрузку

Управление:
W - движение вверх
S - движение вниз  
A - движение влево
D - движение вправо
E - сохранить игру
Q - выход в меню

Компиляция:
bash

g++ main.cpp modul.cpp -o game.exe
./game.exe
